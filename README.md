# g-sig

project G-LocON using WebRTC

## プロトコル protocol

- 共通
    - ping
    - pong
- G-LocON
    - register
    - update
    - delete
    - search
        - static
        - dynamic
    - send
        - WebRTC接続後にP2P通信を行うのでいらないかも。。。？
        - DataProtocolでデータを送るのに使うかもしれない
- P2P
    - offer
        - SDPを送る。
        - WebRTCにおけるoffer側
    - answer
        - SDPを送る。
        - WebRTCにおけるanswer側
    - candidate
        - ICE candidateを受信
    - close
